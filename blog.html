What is the Document Object Model (DOM)?

A programming interface for HTML is called the Document Object Model (DOM). It converts HTML material into a standardized object that can be easily accessed and modified by functional programming languages such as JavaScript. All dynamic web pages rely on the DOM for correct presentation and functionality, as the majority of web page events are handled by non-HTML code.

How do you select an element from the DOM?

One can select an element from the DOM in several ways. Such as.

  • Finding HTML elements by id

    • The easiest way to find an HTML element in the DOM, is by using the element id. (using getElementById).

  • Finding HTML elements by tag name

    • Returns a collection of all elements in the document with the specified tag name. (using getElementsByTagName).

  • Finding HTML elements by class name

    • It shows a collection of all elements in the document with the specified class name. (Using getElementsByClassName).

  • Finding HTML elements by CSS selectors

    • Here we have two CSS selectors.

    • querySelector: The querySelector method allows you to select the first element that matches a specified CSS selector(s) in the document.
    • querySelectorAll: The querySelectorAll method allows you to select all elements that match a specified CSS selector(s) in the document.
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is simply a method for handling events effectively. Instead of adding an event listener to each similar element, we can add one to a parent element and use the event object's.target attribute to initiate an event on a specific target.

                        "const listContainer = document.getElementById('list-container');
                        listContainer.addEventListener('click', function (event) {
                        event.target.parentNode.removeChild(event.target);
                        })"

Here we have a single event listener and a single responding function. In the above-shown method, we have improved the performance, but we have lost access to individual li elements so to resolve this issue, we used a technique called event delegation. The event object has a special property call .target which will help us in getting access to individual li elements with the help of phases.

Event delegation provides the following benefits:
Improved performance: Attaching a single event listener is more efficient than attaching multiple event listeners to individual elements, especially for large or dynamic lists. This reduces memory usage and improves overall performance.
